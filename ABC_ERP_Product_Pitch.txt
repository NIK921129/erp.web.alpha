             ABC INSTITUTE ERP & LMS - THE COMPLETE PLATFORM

Overview:
ABC Institute ERP is a highly advanced, mobile-first Learning Management 
System (LMS) and Enterprise Resource Planning (ERP) platform. Built with 
a premium "Deep Slate / Navy FinTech-style" UI, it delivers a seamless, 
app-like experience for Students, Teachers, and Administrators. 

From zero-friction course enrolment and UPI payment verification to 
real-time chat, AI-powered doubt solving, automated PDF invoicing, 
and deep system analytics—this platform handles it all in one place.

3. TECHNICAL ARCHITECTURE & STACK
========================================================================

FRONTEND (Client-Side)
- Technologies: Vanilla HTML5, CSS3 (Custom Variables), ES6 JavaScript.
- Styling: 100% custom CSS. Zero reliance on bloated frameworks like Bootstrap.
- Layout Strategy: CSS Grid and Flexbox with a custom floating mobile dock.
- Interaction: Asynchronous Fetch API calls provide a fast, Single Page App feel.
- Data Visualization: Chart.js for beautiful, responsive revenue analytics.

BACKEND (Server-Side)
- Core: Node.js and Express.js.
- Database: MongoDB managed via the Mongoose ORM.
- Authentication: Secure, stateless token logic with bcryptjs password hashing.
- File Handling: Multer for managing payment screenshots and assignment uploads.
- Real-Time Engine: Socket.io powers live chat, notifications, and UI updates.
- Communications: Node-Mailjet handles reliable transactional emailing.
- Document Generation: PDFKit dynamically generates automated invoices.
