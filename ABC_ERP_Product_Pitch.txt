ABC Institute ERP is a highly advanced, mobile-first Learning Management 
System (LMS) and Enterprise Resource Planning (ERP) platform. Built with

3. TECHNICAL ARCHITECTURE & METHODS
========================================================================

FRONTEND (Client-Side)
- Tech: Vanilla HTML5, CSS3 (Custom Variables), ES6 JavaScript.
- Styling: Fully custom CSS. No bloated frameworks (Bootstrap/Tailwind).
- Layout: CSS Grid and Flexbox. Mobile UI uses a custom floating dock.
- Interaction: Asynchronous fetch API calls for zero page-reloads (SPA feel).
- Graphs: Chart.js for revenue analytics.

BACKEND (Server-Side)
- Tech: Node.js, Express.js.
- Database: MongoDB via Mongoose ORM.
- Authentication: JWT-like stateless token logic + bcryptjs password hashing.
- File Uploads: Multer (handles payment screenshots & assignment files).
- Real-Time: Socket.io for chat, live notifications, and live UI updates.
- Emailing: Node-Mailjet for transactional emails.
- PDF Generation: PDFKit for automated on-the-fly invoice creation.

CORE API ENDPOINTS & FUNCTIONS:
- POST /api/auth/login : Validates bcrypt passwords, returns token.
- POST /api/payments : Handles multipart/form-data for image uploads.
- POST /api/payments/:id/approve : Core ERP logic (Updates status -> 
  Creates Enrolment -> Updates Course Count -> Generates PDF -> Emails User).
- POST /api/ai/chat : Securely connects to Google Gemini API, deducts 
  user credits, and streams response.
- GET /api/admin/stats : Aggregates platform data (Revenue, user counts).
- POST /api/logs : Middleware-injected logging system capturing x-forwarded-for IPs.
